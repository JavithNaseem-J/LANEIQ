# FreightMind — Multi-Agent Freight Optimization System

Freight brokers and 3PLs in the UAE coordinate shipments reactively via manual processes, with no automated exception handling when port delays or flight disruptions occur.

FreightMind is a multi-agent system that optimizes logistics routing and automates exception handling using LangGraph and OR-Tools.

## Tech Stack
- **Core AI/ML**: LangGraph, OpenAI, OR-Tools
- **API/Backend**: FastAPI, Celery, Redis
- **Frontend**: Streamlit, Plotly
- **MLOps**: MLflow, DVC, Evidently
